# 1.3.2 JS жү§иЎҢжңәеҲ¶жҰӮиҝ°

JavaScript иҜӯиЁҖзҡ„дёҖеӨ§зү№зӮ№е°ұжҳҜ <span style="color: red"> еҚ•зәҝзЁӢ </span>пјҢд№ҹе°ұжҳҜиҜҙпјҢ<span style="color: red"> еҗҢдёҖдёӘж—¶й—ҙеҸӘиғҪеҒҡдёҖд»¶дәӢ </span>гҖӮ

иҝҷжҳҜеӣ дёә Javascript иҝҷй—Ёи„ҡжң¬иҜӯиЁҖиҜһз”ҹзҡ„дҪҝе‘ҪжүҖиҮҙ вҖ”вҖ”JavaScript жҳҜдёәеӨ„зҗҶйЎөйқўдёӯз”ЁжҲ·зҡ„дәӨдә’пјҢд»ҘеҸҠж“ҚдҪң DOM иҖҢиҜһз”ҹзҡ„гҖӮжҜ”еҰӮжҲ‘д»¬еҜ№жҹҗдёӘ DOM е…ғзҙ иҝӣиЎҢж·»еҠ е’ҢеҲ йҷӨж“ҚдҪңпјҢдёҚиғҪеҗҢж—¶иҝӣиЎҢгҖӮ еә”иҜҘе…ҲиҝӣиЎҢж·»еҠ пјҢд№ӢеҗҺеҶҚеҲ йҷӨгҖӮ

еҚ•зәҝзЁӢе°ұж„Ҹе‘ізқҖпјҢжүҖжңүд»»еҠЎйңҖиҰҒжҺ’йҳҹпјҢеүҚдёҖдёӘд»»еҠЎз»“жқҹпјҢжүҚдјҡжү§иЎҢеҗҺдёҖдёӘд»»еҠЎгҖӮиҝҷж ·жүҖеҜјиҮҙзҡ„й—®йўҳжҳҜпјҡ еҰӮжһң JS жү§иЎҢзҡ„ж—¶й—ҙиҝҮй•ҝпјҢиҝҷж ·е°ұдјҡйҖ жҲҗйЎөйқўзҡ„жёІжҹ“дёҚиҝһиҙҜпјҢеҜјиҮҙйЎөйқўжёІжҹ“еҠ иҪҪйҳ»еЎһзҡ„ж„ҹи§үгҖӮ

# 1.3.3 еҗҢжӯҘе’ҢејӮжӯҘзҡ„жҰӮеҝө

дёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢеҲ©з”ЁеӨҡж ё CPU зҡ„и®Ўз®—иғҪеҠӣпјҢHTML5 жҸҗеҮә Web Worker ж ҮеҮҶпјҢе…Ғи®ё JavaScript и„ҡжң¬еҲӣе»әеӨҡдёӘзәҝзЁӢгҖӮдәҺжҳҜпјҢJS дёӯеҮәзҺ°дәҶеҗҢжӯҘе’ҢејӮжӯҘгҖӮ

  • еҗҢжӯҘ

    еүҚдёҖдёӘд»»еҠЎз»“жқҹеҗҺеҶҚжү§иЎҢеҗҺдёҖдёӘд»»еҠЎпјҢзЁӢеәҸзҡ„жү§иЎҢйЎәеәҸдёҺд»»еҠЎзҡ„жҺ’еҲ—йЎәеәҸжҳҜдёҖиҮҙзҡ„гҖҒеҗҢжӯҘзҡ„гҖӮжҜ”еҰӮеҒҡйҘӯзҡ„еҗҢжӯҘеҒҡжі•пјҡжҲ‘д»¬иҰҒзғ§ж°ҙз…®йҘӯпјҢзӯүж°ҙејҖдәҶпјҲ10 еҲҶй’ҹд№ӢеҗҺпјүпјҢеҶҚеҺ»еҲҮиҸңпјҢзӮ’иҸңгҖӮ

  • ејӮжӯҘ

    дҪ еңЁеҒҡдёҖд»¶дәӢжғ…ж—¶пјҢеӣ дёәиҝҷд»¶дәӢжғ…дјҡиҠұиҙ№еҫҲй•ҝж—¶й—ҙпјҢеңЁеҒҡиҝҷд»¶дәӢзҡ„еҗҢж—¶пјҢдҪ иҝҳеҸҜд»ҘеҺ»еӨ„зҗҶе…¶д»–дәӢжғ…гҖӮжҜ”еҰӮеҒҡйҘӯзҡ„ејӮжӯҘеҒҡжі•пјҢжҲ‘д»¬еңЁзғ§ж°ҙзҡ„еҗҢж—¶пјҢеҲ©з”Ёиҝҷ 10 еҲҶй’ҹпјҢеҺ»еҲҮиҸңпјҢзӮ’иҸңгҖӮ

<span style="color: red"> д»–д»¬зҡ„жң¬иҙЁеҢәеҲ«пјҡ иҝҷжқЎжөҒж°ҙзәҝдёҠеҗ„дёӘжөҒзЁӢзҡ„жү§иЎҢйЎәеәҸдёҚеҗҢгҖӮ</span>

# 1.3.4 еҗҢжӯҘд»»еҠЎе’ҢејӮжӯҘд»»еҠЎ

  • еҗҢжӯҘд»»еҠЎ

    ејӮжӯҘд»»еҠЎзӣёе…іж·»еҠ еҲ° <span style="color: red"> д»»еҠЎйҳҹеҲ— </span > дёӯпјҲд»»еҠЎйҳҹеҲ—д№ҹз§°дёәж¶ҲжҒҜйҳҹеҲ—пјүгҖӮ

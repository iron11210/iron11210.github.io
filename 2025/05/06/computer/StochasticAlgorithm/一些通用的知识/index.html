# Voronoi 图复杂度的欧拉公式证明

我们希望证明：对于 n>2n > 2n>2 个点的 Voronoi 图 V(P)V(P)V(P)，其复杂度是线性的：

  • 顶点数 ≤2n−5\leq 2n - 5≤2n−5
  • 边数 ≤3n−6\leq 3n - 6≤3n−6

# 📐 使用欧拉公式

欧拉公式是平面图中的基本定理：

V−E+F=2V - E + F = 2V−E+F=2

其中：

  • VVV 是顶点数
  • EEE 是边数
  • FFF 是面数（Voronoi cell 数量，即原始点数 nnn）

# 🔧 添加虚拟点使图封闭

由于 Voronoi 图的某些边是无穷延伸的半线，不能直接套用欧拉公式。为了解决这个问题，我们：

  • 添加一个虚拟点 vvv
  • 将所有半线连到 vvv，使图成为封闭图

这样我们有一个新的图，满足欧拉公式：

(V+1)−E+n=2（记作 *）(V + 1) - E + n = 2 \quad \text{（记作 *）}(V+1)−E+n=2（记作 *）

# 🔁 顶点度数下界

Voronoi 图中的每个顶点（包括新增的 vvv）至少有 度数为 3，因为：

  • 一个 Voronoi 顶点是三个 Voronoi cell 的交点
  • 所以它至少连接三条边

因此：

  • 所有顶点度数之和 ≥3(V+1)\geq 3(V + 1)≥3(V+1)
  • 而总度数也等于 2E2E2E

所以：

2E≥3(V+1)⇒V≤23E−12E \geq 3(V + 1) \Rightarrow V \leq \frac{2}{3}E - 12E≥3(V+1)⇒V≤32​E−1

# 🔚 代入欧拉公式

将上式代入公式 (*)：

(23E−1+1)−E+n=2⇒23E−E+n=2⇒−13E=2−n⇒E=3n−6\left(\frac{2}{3}E - 1 + 1\right) - E + n = 2 \\ \Rightarrow \frac{2}{3}E - E + n = 2 \\ \Rightarrow -\frac{1}{3}E = 2 - n \\ \Rightarrow E = 3n - 6(32​E−1+1)−E+n=2⇒32​E−E+n=2⇒−31​E=2−n⇒E=3n−6

再代入 V≤23E−1V \leq \frac{2}{3}E - 1V≤32​E−1 得：

V≤23(3n−6)−1=2n−5V \leq \frac{2}{3}(3n - 6) - 1 = 2n - 5V≤32​(3n−6)−1=2n−5

# ✅ 结论

  • 边数 E≤3n−6E \leq 3n - 6E≤3n−6
  • 顶点数 V≤2n−5V \leq 2n - 5V≤2n−5
  • Voronoi 图总结构复杂度为 O(n)O(n)O(n)（线性）

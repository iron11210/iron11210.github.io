# 1. 为什么需要？

这个性能工具的作用是，能够让我们自己知道，每个阶段分别耗时多少，并且哪个阶段最慢，用了哪些步骤。

# 2. 核心

这个轻量级性能分析工具的本质，其实就是一个树形结构的计时器。每个节点都是一个 Profiler 的实例，可以

  • 记录这段任务的耗时
  • 记录拥有哪些子节点
  • 输出统计结果

unity无法识别script错误

在开发的过程中，碰到了一个很奇怪的问题 而且是本来能够挂载到对象上的 script 忽然报错显示挂载失败，虽然修改了程序中的一些可能造成报错的代码后挂载成功，但是还是很奇怪 实际上原因是因为 unity 自身加载失败的问题，把 library 文件删了再重新启动一遍项目即可解决。 同时可能存在的其他问题：如果 script 自身有编译错误或者类名和文件名不符也可能会产生这个报错。
